March 01 2018

IE 盒模型与 W3C 盒模型

  • W3C 盒模型，即，content-box 的宽度从 Content Edge 开始算起，也就是当有 padding 跟 border 的时候，实际宽度会扩大
  • IE 盒模型，即，border-box 的宽度从 Padding Edge 开始算起，也就是 width = content-width + padding-left + padding-right + border-left + border-right，当有 padding 跟 border 的时候，实际宽度并不会扩大

  • content-box：页面将采用 W3C 盒模型来解析计算，默认值
  • border-box：页面将采用 IE 盒模型解析计算
  • inherit：页面将从父元素继承 box-sizing 的值

border-box

You might think of it this way: with box-sizing: border-box the padding and border press their way inside the box rather than expand the box. The result is a box the exact width you set it to be and can count on.

为了计算和方便（不管设置怎样的 border 和 padding，盒子的实际宽度都不变），一些专家建议使用 border-box
