和弦初探

January 26 2018

## 写在前面

下面均以根音 C 的和弦为例 如果还没弄清楚音程的概念，请看我之前的文章 ===> 音程

下面出现的大写字母 C 表示C的大三和弦，Cm 表示小三和弦；而 major 用来修饰 7 度音，一般表示七和弦中的 7 音与根音为大七度，没有 major 修饰为小七度。小括号表示需要特别标注的音，下面提及到的是特别针对 5 音的升降。aug 和 dim 分别表示增和弦和减和弦，规律可能有些诡异，要特别记忆。

## 三和弦

和弦名称 公式 组成音 示例
大三和弦 大三度 + 小三度 1-3-5 C
小三和弦 小三度 + 大三度 1-b3-5 Cm
增三和弦 大三度 + 大三度 1-3-#5 Caug
减三和弦 小三度 + 小三度 1-b3-b5 Cdim

## 基本七和弦

和弦名称 公式 组成音 示例
大七和弦 大三和弦 + 大七度 1-3-5-7 Cmaj7
小七和弦 小三和弦 + 小七度 1-b3-5-b7 Cm7
属七和弦 大三和弦 + 小七度 1-3-5-b7 C7
减七和弦 减三和弦 + 减七度 1-b3-b5-bb7 Cdim7
半减七和弦 减三和弦 + 小七度 1-b3-b5-b7 Cm7(b5)

## 变化七和弦

和弦名称 公式 组成音 示例
大七增五和弦 大七和弦 + #5 1-3-#5-7 Cmaj7(#5)
小七增五和弦 小七和弦 + #5 1-b3-#5-b7 Cm7(#5)
小调大七和弦 小三度 + 大七度 1-b3-5-7 Cmmaj7
小调大七减五和弦 小调大七 + b5 1-b3-b5-7 Cm(maj7-5)

## 延伸和弦

和弦名称 公式 组成音 示例
九和弦 属七和弦 + 大九度 1-3-5-b7-9 C9
大九和弦 大七和弦 + 大九度 1-3-5-7-9 Cmaj9
小九和弦 小七和弦 + 大九度 1-b3-5-b7-9 Cm7
十一和弦 九和弦 + 完全十一度 1-3-5-b7-9-11 C11
大十一和弦 大九和弦 + 完全十一度 1-3-5-7-9-11 Cmaj11
小十一和弦 小九和弦 + 完全十一度 1-b3-5-b7-9-11 Cm11
十三和弦 十一和弦 + 大十三度 1-3-5-b7-9-11-13 C13
大十三和弦 大十一和弦 + 大十三度 1-3-5-7-9-11-13 Cmaj13
小十三和弦 小十一和弦 + 大十三度 1-b3-5-b7-9-11-13 Cm13

## 分数和弦

- 转位和弦：C/E --> Bass 为和弦內音 - 复合和弦：G/F --> Bass 为和弦外音，和弦组成音为 F-G-B-D
