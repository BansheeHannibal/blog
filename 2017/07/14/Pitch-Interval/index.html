音程

July 14 2017

写在前面

没有错，这一部分要说的就是音程。平时学习音乐在网上会听到有人说什么小三度啊，五度循环圈？？？每当听到这些词都很困惑，这到底是什么鬼呢，究竟有什么用。初中的时候我们就开始学物理，我们想要知道我们的桌子有多长，抽象点说，就是桌子的一边到另一边的距离，我们会用这个词来描述这个长度，而单位呢就是我们常常用的米(m)，分米(dm)，厘米(cm)；而描述两个音之间的距离，我们用音程来描述，而它也有相应的单位，那就是度。

度

度，就是从一个音数到另一个音，数了多少个，就是多少度

  • C –> C 1度
  • C –> D 2度
  • C –> E 3度
  • C –> F 4度
  • C –> G 5度
  • C –> A 6度
  • C –> B 7度
  • C –> C 8度

注：需要特别注意的是，C –> #D 也是 2度，C –> bG 也是 5度。无论怎样升怎样降，判断度数只需要看音名，然后数音名就可以了。

加个形容词

描述音程不仅仅是用多少度来表示，前面还需要加一个形容词，这些形容词可以是，纯、小、大、减、增，为什么呢，就像上面一点提到的 C –> D 是 2度，C –> #D 也是 2度，他们同样是 2度 ，为了区别他们，我们就在他们前面加上不同的形容词。

下面要说的是重点了，要看清楚了。

  • 1 4 5 8 纯、增、减
  • 2 3 6 7 大、小、增、减

意思是，度数为 1 4 5 8 前面的形容词只有纯、增、减三种，度数为 2 3 6 7 前面的形容词只能是大、小、增、减；不会出现纯二度，也不会出现大五度的说法的哦。

判断它是什么音程

判断两个音之间的音程有几个步骤：

  • 数一数他们有多少个度
  • 判断是 1458 还是 2367
  • 再用半音数与纯音程或者大音程比较，比纯音程音数少的，就是减，反之就是增音程；比大音程的半音数少的就是小音程，比小音程更少的就是减音程，那么比大音程半音数多的当然就是增音程喇。

听完上面几点步骤还是一头雾水，不要紧，下面我们详细而简明地分析一下是如何判断出来的。

音程的本质是由两个音之间的半音数决定的，我们只是给它取不同的名字，方便区分他们。在分析之前，我们先了解一下十二平均律。

  • 十二平均律
    十二平均律就是音之间的全半关系，只需要记住一般两个音之间是全音关系，E和F 是半音关系，B和C 也是半音关系。这就是为什么在键盘上，E和F 之间没有黑键，B和C 之间也没有黑键。

    举个栗子就很好懂喇。C 和 G 之间相差 7 个半音

  • 2 3 6 7 度
    我们先来看看标准的大二度 看回上面的键盘，C 和 D 之间相差 2个半音，我们把它称为大二度。我们再来看一个 , E 和 F 相差 1个半音，比大二度少一个半音，所以我们称它为小二度。
    然后我们来看看标准的大六度是怎样子的。，数半音的时候脑海中要有一个键盘的画面，数完之后发现大六度有 9个半音；没错，这时候我们来看看小六度。，回想一下键盘，E和C 之间相差8个半音，所以它就叫做小六度咯。
    这时候你会纳闷😕，难道我每一次都要去数他们有多少个半音再比较吗？其实，你熟悉键盘的音的大概位置，你就能很快很准确的判断出音程的名字。方法就是，首先记住标准的大音程的样子，我们给E和F，B和C 这种之间没有黑键的结构，我们自己取名字叫做半音结构，因为包含半音结构，肯定比不包含半音结构的音程的半音数要少的。标准的大二度不包含半音结构，标准的大三度也不包含，标准的大六度和大七度包含一个半音结构；我们就可以根据，如果一个2度（或3度）音程包含一个半音结构，就叫做小二度（或小三度）；如果一个6度（或7度）音程包含两个半音结构，就叫做小六度（或小七度）

  • 1 4 5 8 度
    标准的纯四度是这样的，我们不再数半音了，因为太繁琐了，我们只需要判断它是否包含上面一点提到的半音结构。标准的纯四度包含一个半音结构，那么不包含半音结构的4度音程，就叫做增四度了，比如这个。说了这么多，这个一看就是纯五度喇。标准的纯八度包含两个半音结构。纯一度就不用说喇，就是自己。
    再说一次 ☝ 包含半音结构的比不包含半音结构的，半音数要少一个。我们要记住标准的大二度，大三度，大六度，大七度，纯四度，纯五度的样子，再根据是否包含，或包含多少个半音结构来与标准的音程的半音数比较，比大更大，就是增，比小更小，就是减；比纯大，就是增，比纯小，就是减。

最后看两个比较难的栗子

第一个栗子

  • D 和 B 是 6度 音程，包含一个半音结构，与标准大六度一致，所以是大六度
  • E 和 C 是 6度 音程，包含两个半音结构，比标准大六度少一个半音，所以是小六度
  • G 和 C 是 4度 音程，包含一个半音结构，与标准纯四度一致，所以是纯四度
  • D 和 E 是 2度 音程，不包含半音结构，与标准大二度一致，所以是大二度
  • E 和 F 是 2度 音程，包含一个半音结构，比标准大二度少一个半音，所以是小二度

第二个栗子

  • D 和 G 是 4度音程，包含一个半音结构，与纯四度一致，但是 D 降了一个半音，即音程多了一个半音，距离变宽，所以是增四度
  • A 和 C 是 3度音程，包含一个半音结构，比标准大三度少一个半音，所以是小三度，又因为 A 升了一个半音，距离变窄，所以是减四度
  • E 和 C 是 6度 音程，包含两个半音结构，比标准的大六度少一个半音，所以是小六度，因为 E 降了一个半音，距离变宽，所以是大六度
  • B 和 F 是 5度 音程，包含两个半音结构，比标准的纯五度少一个半音，所以是减五度
  • F 和 D 是 6度 音程，包含一个半音结构，与标准的大六度一致，又因为 F 升了一个半音，D 降了一个半音，距离变窄了两次，所以变成了减五度

音程转位

音程转位的概念很简单哦，就是将根音升高8度，转位后的音程，名字中的增变成减，减变成增，大变成小，小变成大，数字相加等于9就好了。

比如图中的左边就是大二度，转位后就是右边，变成小七度。
