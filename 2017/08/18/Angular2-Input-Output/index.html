Angular2 父子组件的通信

August 18 2017

父组件向子组件传入数据 – @Input

一个组件的标签在另一个组件的内部，即包含这个标签的组件就是它的父组件，子组件的属性通过加上 @Input() 装饰器接收来自父组件的数据。

child.component.ts

father.component.ts

在父组件通过中括号 [ ] 单向绑定属性 f_Profess 和 fItem 到子组件。子组件通过加上 @Input 装饰器接收来自父组件的数据。但是要注意的是，基本数据类型是单向绑定，引用类型的变量是双向的，fItem 属性就是引用类型，子组件 item 属性的改变，会影响到父组件，使用时需谨慎。下图，改变输入框的值，也就是改变子组件的 item 的值，父组件对应的值也随之改变，但基本数据类型 f_Profess 不会影响到父组件。

子组件向父组件传出数据 – @Output

在子标签添加一个事件 (pollux)，名字随意，然后在父组件下面写一个监听的函数 onChange(event)。

但这个监听是没有意义的，这个时候要在子组件用 @Output 标签来获得与父组件的联系。

这个时候只需要调用 EventEmitter 类的发射函数 emit() 就能把数据发送到父组件，父组件通过 onChange(event) 函数接收。
